KAPieChart
==========

Basic pie chart for iOS.

Features:
- Any size chart.
- Unlimited slices.
- Any colors.
- Any size slices.
- Easy to implement.
- Fully vectorized (no bitmap images).
- Any slice can be ‘popped out’ to create separation from chart
- Fast.
